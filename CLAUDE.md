# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This is a learning project containing browser-based games and tools built as self-contained HTML files (HTML + CSS + JS in a single file, no build step or dependencies).

## Running the Project

Open any HTML file directly in a browser:

```bash
open tictactoe.html
```

## Architecture

Each project is a **single self-contained HTML file** — no framework, no bundler, no package manager. All HTML, CSS, and JavaScript live in one file.

## Git Workflow

- Every change gets a clean, descriptive commit and is pushed to GitHub: https://github.com/memontgo90-tech/ClaudeCodeLearn
- Commit messages should summarize *what* changed and *why* in 1–2 sentences.
- Push after every commit: `git push`
